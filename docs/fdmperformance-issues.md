# 智能绩效问题与优化清单

## 已处理

- 用户下拉框搜索异常：考评表基础信息、流程设计器指定成员支持按姓名、用户名、部门、岗位模糊搜索，并按“姓名 · 部门 · 岗位”展示。
- 考评表更新时间显示时间戳：考评表列表统一格式化更新时间。
- 新建考评表默认带入当前/首个用户：新建时被考核人改为空，需要人工选择。
- 流程节点校验文案暴露字段名：`assignEmptyHandlerType`、`assignStartUserHandlerType` 改为业务提示。
- 指标候选项误解：考核指标页增加提示，说明只有勾选的指标才会保存；保存校验文案改为“请先勾选要参与考核的指标，或在维度下新增/导入指标”。
- 发起考核页缺少流程确认信息：预览和确认弹窗增加流程摘要，便于核对主管评分人。
- 控制台无效路由：补充 `/fdmperformance/mock-workbench/index.vue` 兼容组件，避免旧菜单组件路径导致路由报错。
